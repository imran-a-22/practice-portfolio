How It Works

Bubble sort repeatedly compares adjacent elements and swaps them if they're in the wrong order. Each pass "bubbles" the largest element to the end. It's simple but inefficient for large datasets.

When To Use

Use bubble sort only for educational purposes or when the array is nearly sorted. Its O(n²) complexity makes it impractical for real-world large datasets. Nearly-sorted arrays can achieve O(n).
